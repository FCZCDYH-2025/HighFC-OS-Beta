HighFC OS Beta
🚀 项目简介

HighFC OS‌ 是一款开源、免费的个人操作系统项目，致力于提供‌方便快捷的交互体验‌与‌清新靓丽的视觉设计‌。系统以“‌虽然冷门，但是硬核‌”为理念，融合了现代化交互与经典拟物化风格，并持续进行功能迭代与体验优化。

当前仓库为 ‌HighFC OS Beta‌，正在全面重构，预计于 ‌2026年7月3日‌ 发布首个重构版本。

🔧 功能亮点‌

内置浏览器、电话（雏形）、时钟、电量显示
支持网络随机壁纸与在线音乐播放
集成 AI 功能（基于 Ollama，需自行安装模型）
扩展支持（.she 扩展框架）
蓝牙连接支持（通过 WebBluetooth，需在 Chrome 中加载 HTML 体验）
通知历史记录与网络图片屏保

📦 系统更新与支持‌

持续版本迭代，修复优化不断
部分历史版本已开源（详见下文“已开源版本列表”）
提供真正的系统更新机制与更新日志追溯
